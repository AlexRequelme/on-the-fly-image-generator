## Description

On the fly image generator with NestJs

## Running the app

```bash
# development
$ pnpm run dev

# watch mode
$ pnpm run dev:debug

# production mode
$ pnpm run start:prod
```

## Preview

Go to http://localhost:5000/api/dynamic-image
